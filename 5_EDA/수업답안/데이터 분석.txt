데이터 분석 이해

*데이터 분석이란?
- 주어진 문제를 해결하기 위해 데이터를 다루는 과정
- 현상의 이해를 통해 미래를 예측하는 문제해결기법
- 인사이트를 도출하기 위해 알고리즘과 수학적 처리과정을 적용하여 해당 정보에
대한 결론을 도출하고 패턴을 차는 과정

분석대상선정 -> 분석과제 수행 -> 시스템/프로세스 개선 -> 모니터링

*분석과제 수행 (자동화)
- 문제정의 데이터 수집 
- 데이터 정제/가공 
- 탐색적 데이터 분석 
- 모형적합 및 평가 
- 시각화 및 리포팅

* 스몰데이터 분석
- 분석 목적: - 표본분석을 토대로 모집단의 특성을 추론하여 해석함
               - 비교집단간 특성차이, 조치수단 간 효과차이 비교를 통해
                 관심대상 집단/수단을 선별하는데 중점
- 데이터 수집: 사용자를 대상으로 한 서베이나 포커스그룹 인터뷰 활용으로
                   상당한 조사기간이 수반됨
- 분석기법: 기술통계 및 차이분석, 회귀분석, 구조방정식 등 추론통계 중심
- 분석도구: 전통적 SPSS, SAS 등 상용 통계분석 패키지 활용

* 빅데이터 분석
- 분석목적: - 대규모 데이터에 숨어있는 패턴을 발견하고 규칙을 도출함
               - 도출된 패턴 및 규칙을 이용해 개별대상별(고객/제품 등) 액션방안 마련에 중점
- 데이터 수집: 대규모 내부거래처 데이터, 음성, 동영상, 오

시티즌 데이터 사이언티스트

* 데이터 분석의 목적
- 다양한 내/외부, 정형/비정형 데이터를 획득하여 새로운 통찰과 가치 창출
- 업무에 대한 문제해결 및 의사결정

* 데이터분석을 학습하기 위한 방법
- 데이터분석기법중심
- 분석도구 중심
- 문제해결 중심

* 분석대상: 데이터분석 대상은 세상에 존재하는 다양한 유무형의 객체들이 가지고 있는 속성과 관계들

* 다양한 분석 자원 추출
- 분석 대상 객체로부터 분석에 필요한 속성, 변수 추출
- 속성/변수 리스트
인구 사회학적 속성
거래 이력 속성
라이프 스타일 속성
성향 성격 속성

* 문제와 데이터 분석
- 탐색 기술분석 -> 문제의 중요도 인식
- 설명 인과분석 -> 문제 전개과정의 규명
- 최적 처방분석 -> 문제 처리방안의 도출

* 분석과제 기획 방향
- 수요기반 분석과제 도출
- 데이터주도 분석과제 도출

문제인식 -> 자료리뷰 -> 데이터모델링(가설수립) -> 데이터 수

* 데이터 분석 과정: 주어진 과제에 대하여 필요한 데이터를 분석하고 최종 결론을 이끌어 내는 과정
                       데이터 분석 및 절차 / 데이터 실험 / 데이터 분석의 시각화 /휴리스틱 분석
* 통계분석:
데이터수집 및 특성파악 -> 히스토그램 -> 데이터의 특성 상세화 -> 분할표 작성
-> 독립성, 적합도 검정 -> 차이 검정 -> 상관(인과) 관계

* 통계분석 및 응용 및 확장: 요인분석, 주성분분석, 정준상관분석, 구조방정식 모형,
                                    시계열분석, 판별분석, 텍스트마이닝, 일반화 선형모델,
                                    로지스틱 회귀, 결정트리, 나이브베이즈분류기, 군집분석, 앙상블

# 기본 용어 및 개념
* 모집단과 표본
- 모집단(population): 조사 대상이 되는 관측 가능한 개체로 된 집단 전체 -> 모수(parameter)
- 표본(sample): 모집단에서 선택된 모집단 구성단위의 일부 -> 통계량(statistic)
기술통계: 자료의 정리, 요약, 분석
추특통계: 추정/검정

* 전수조사와 표본조사
- 전수조사: 모집단 전체를 조사
- 표본조사: 모집단 일부를 조사

* 모수와 통계량
- 모수: 모집단의 특성을 나타내는 수치
- (표본)통계량:

- 시각화도구
- 기술통계량

  추측하는 기술

# 데이터 (data)
- 가치 판단의 근거가 될 수 있는 것
- 현실 세계로부터의 단순한 관찰이나 측정을 통해 수집된 사실이나 값
- 대상물 또는 상황
- 원자료 (raw data) : 가공되지 않은 상태
- 분석의 원자재

* Dataset
- 관찰대상으로부터 속성들을 측정한 자료들의 모임
행: 관측값(개체)
열: 속성, 변수(측정, 자료)

* 변수
마케팅 비용, 프로모션 기간
X변수: 독립변수, 설명변수, 예측변수
매출액
Y변수: 종속변수, 반응변수, 결과변수

* 정성적 데이터
- 비정형 데이터
- 객체 하나에 함축된 정보를 가짐
- 언어, 문자
- 파일, 웹
- 외부 시스템 (소셜 데이터)

* 정량적 데이터
- 정형, 반정형 데이터
- 여러 속성이 모여 객체를 이룸
- 수치, 도형, 기호
- 데이터베이스, 스프레드시트,
- 내부시스템(RDBMS,

* 범주형 데이터
- 정해진 유형이나 순서로 구분되는 데이터
- 명복형: 분류를 목적으로 하는 데이터 (크기 비교 불가능)
- 순서형: 대, 중, 소와 같이 값을 수서로 분류할 수 있는

* 숫자형 데이터
- 데이터 자제가 숫자로 표현
- 데이터 속성을 그대로 반영
- 이산형: 셀 수 있는 값을 나타내는 데이터
- 연속형: 측정 대상의 크기 변화가 연속적인 데이터

* 척도: 데이터를 측정하기 위해 임의로 부여한 숫자 간의 관계
- 명목척도: 명목형 자료, 분류
               예) 성별
- 서열(순서)척도: 순서형 자료, 서열 결정
               예) 
- 구간(등간)척도: 자료의 각 값이 동일한 간격으로 이뤄진 자료
                       상대적 순위 측정, 상대영점
- 비율척도: 다른 것보다 몇배 더 큰가를 나타낼 수 있는 척도
               절대 영(0)점이 존재(절대 영점은 값의 크기가 0), 사칙연산이 가능
               상대적 크기, 간격, 몇 배
